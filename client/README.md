# Traffic Cams Client

This package contains the browser UI for `traffic-cams`.

## Scripts

- `npm run dev` starts the live client for local development
- `npm run dev:demo` starts the reproducible demo path used for screenshots
- `npm test` runs the client-side unit tests
- `npm run build` creates the production build

## Demo Review

Use the demo path when you want to inspect the UI without live feeds or private map configuration.

- overview: `http://127.0.0.1:4173/?demo=1`
- inspector state: `http://127.0.0.1:4173/?demo=1&selected=london-01`

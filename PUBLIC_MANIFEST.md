# Public Manifest

## Included

- `client/`
- `server/`
- `docs/`
- `.env.example`
- `.gitignore`
- `LICENSE`
- `README.md`
- product and reference markdown that is already public-safe

## Excluded

- `.git/`
- `node_modules/`
- `dist/` and build output
- local `.env` files
- logs, pid files, DB files, and caches
- quarantined workspace material outside this repo directory
